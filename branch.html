文件结构

HDL系统框架是以jQuery为基础的。所以页面必须引入jquery.js文件，1.8以上版本。
新建的页面放在根目录下，或在根目录下新建文件夹存放。使用时注意路径。
根目录下的common文件夹存放系统通用文件，不要改变路径和文件夹名称，包括里面的文件，否则使用时您要做相应的路径改动。
以下是common文件结构：

common
  |- css               //存放所有样式文件
      |- config.css    //系统样式（必须）
  |- data              //存放固定数据的文件，这里一般放 json 文件，通过ajax调用
  |- images            //图片文件夹
  |- jedata            //日期插件文件夹
  |- js                //存放js文件
      |- config.js     //系统js（必须）
      |- divscroll.js  //左侧菜单纵向滚动条样式控制js
      |- jquery.jqprint-0.3.js  //jqprint打印插件
      |- jquery.min.js  //jquery库文件（必须）
      |- jquery-form.js //jquery表单js，特别当表单提交包含文件上传时用到。
      |- jquery-migrate-1.1.0.js  //jqprint打印的IE兼容文件,与jqprint.js一起使用。
      |- store.js      //本地存储插件
      |- 自定义自己的js文件
